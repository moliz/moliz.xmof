For running the xMOF projects the following Eclipse Plugins have to be installed:
* Graphiti
* Xtend (not required if code generation not necessary)

The following additional projects are required:
* EMF Profiles (checkout at https://code.google.com/a/eclipselabs.org/p/emf-profiles/)

For the examples the following Plugins are additionally required:
* GMF (Editor Support)
